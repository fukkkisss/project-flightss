# project-flightss
Проект, в котором исследуются тенденции в авиаиндустрии, а именно способность авиакомпаний восстанавливаться после крупных шоков и возвращаться в прежний ритм. Рассмотрены также факторы, которые влияют на скорость восстановления компании&

Единственный входной файл — flights_sample_3m.csv (~3 млн рейсов). При первом запуске он конвертируется в колоночный flights.parquet средствами DuckDB (потоково, со сжатием ZSTD); при повторных запусках шаг конвертации пропускается.
